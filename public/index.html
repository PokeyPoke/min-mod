Alpha Vantage API (Stocks)

Get real-time stock market data. Free tier includes 25 calls/day, 5 calls/minute.

The Odds API (Sports)

Get live sports odds and betting data. Free tier includes 500 requests/month.

Get your free key: the-odds-api.com

About API Keys

  • API keys are stored locally in your browser
  • Crypto data doesn't require an API key (uses CoinGecko free tier)
  • All other widgets work without API keys
  • Setting up APIs enables real-time, accurate data
